HTML5 Facial Recognition Demo

HTML5 Heroku hosted demo which will recognize an image taken from a Webcam, associate it with a user's name and then allow any image of the same person taken again with a Webcam to be recognized as that user. Or another user that has already been created.

 Get Started

What does it do?

  1. Takes an image of a user from a Webcam and the user's name from a form.
  2. Adds the image to the ReKognition cloud with the associated user's name using the Face::Add REST API method. This method can also determine if the image has a face or not.
  3. Analyzes the image to create facial related data using the Face::Train REST API method. This data is associated to the user's name as a tag.
  4. Takes an additional image of the user with a Webcam, then uses the Face::Recognize REST API method to match the image with an existing user's name.

ReKognition REST API

HTML5 based technologies

WebRTC
I tweaked the awesome Webcam angular.js directive from Jonas Hartmann.
Canvas
I created an angular.js directive which is dependant on the Webcam directive to capture Webcam snapshots. Inspiration and some code shamelessly ripped off from the Webcam angular.js directive's Take Snapshot demo source.
